# `genesis tasks flow`

Flow commands are subcommands of `genesis tasks`, not a standalone `flows` command.

```bash
genesis tasks flow list [--json]
genesis tasks flow show <lookup>
genesis tasks flow cancel <lookup>
```

For full documentation see [Task Flow](/automation/taskflow) and the [tasks CLI reference](/cli/tasks).
